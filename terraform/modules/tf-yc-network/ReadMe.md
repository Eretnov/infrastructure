# Terraform Module: yandex_cloud_network

Этот модуль Terraform получает данные о существующей VPC-сети и подсетях (subnet) по заданным зонам доступности Яндекс Облака.

## Описание

- Использует провайдер [Yandex.Cloud](https://registry.terraform.io/providers/yandex-cloud/yandex/latest/docs)
- Получает существующую VPC-сеть
- Ищет подсети в указанных зонах доступности

## Переменные

| Имя              | Тип           | Описание                                  | Значение по умолчанию         |
|------------------|---------------|-------------------------------------------|-------------------------------|
| network_zones    | list(string)  | Список зон, в которых искать подсети      | `["ru-central1-a", "ru-central1-b", "ru-central1-d"]` |

## Требования

- Terraform = 1.5.7
- Провайдер Yandex Cloud версии >= 0.87.0
